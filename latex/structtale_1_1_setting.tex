\doxysubsection{Detailed Description}
Stores all settings necessary for the simulation. 

Only one \mbox{\hyperlink{structtale_1_1_setting}{Setting}} object should exist per simulation, so its run can be reproduced using the same \mbox{\hyperlink{structtale_1_1_setting}{Setting}}.
