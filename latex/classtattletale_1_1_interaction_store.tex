\doxysubsection{Detailed Description}
Stores Prototypes for all possible interactions. 

An \mbox{\hyperlink{classtattletale_1_1_interaction}{Interaction}} should only be created through this class, so it\textquotesingle{}s field are correctly filled. Also contains helper function to get appropriate \mbox{\hyperlink{classtattletale_1_1_interaction}{Interaction}} options for each situation
