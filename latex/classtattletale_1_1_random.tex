Getter for a random index according to a probability distribution. 

Returns a random index for the passed vector, picked according to the chances stored in said vector (between 0.\+0f and 1.\+0f). If all passed values are zero a division through zero happens, because of that we have to check for that case. As this is quite a costly operation though, the user has to turn that check on manually if they know it could happen with the bool parameter. If it is turned on, all zeroes would mean every index has the same chance to be picked.
