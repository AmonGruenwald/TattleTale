Calculates the chance of an \mbox{\hyperlink{classtale_1_1_interaction}{Interaction}} based on its \mbox{\hyperlink{structtale_1_1_tendency}{Tendency}} and the \mbox{\hyperlink{classtale_1_1_actor}{Actor\textquotesingle{}s }} current state. 

Every value used is based between -\/1.\+0 and 1.\+0 except the return value which is between 0.\+0 and 1.\+0. Because the \mbox{\hyperlink{classtale_1_1_actor}{Actor}} uses this value for a distribution to pick the \mbox{\hyperlink{classtale_1_1_interaction}{Interaction}} it wants to do, 1.\+0 does not automatically mean an \mbox{\hyperlink{classtale_1_1_interaction}{Interaction}} has a 100\% chance of being picked. If we have for example three \mbox{\hyperlink{classtale_1_1_interaction}{Interactions }}, which prodcue the retun values 1.\+0, 1.\+0 and 0.\+0. Then the first two Interactions have a chance of 50\% each and the last one has a 0\% chance.
