\doxysubsection{Detailed Description}
A \doxylink{classtattletale_1_1_course}{Course} grouping \doxylink{classtattletale_1_1_actor}{Actors } together during different slots of the day. 

A \doxylink{classtattletale_1_1_course}{Course} happens a few times week. Those times are called slots. Each slots has a fixed group of \doxylink{classtattletale_1_1_actor}{Actors } that visit the \doxylink{classtattletale_1_1_course}{Course} during that time. As \doxylink{classtattletale_1_1_actor}{Actors } can have the same \doxylink{classtattletale_1_1_course}{Course} a few times each week, some slots feature the same group of \doxylink{classtattletale_1_1_actor}{Actors } but it is guaranteed that \doxylink{classtattletale_1_1_actor}{Actors } always visit a single \doxylink{classtattletale_1_1_course}{Course} in the same group.
