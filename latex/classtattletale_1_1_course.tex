\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classtattletale_1_1_course}{Course}} grouping \mbox{\hyperlink{classtattletale_1_1_actor}{Actors }} together during different slots of the day. 

A \mbox{\hyperlink{classtattletale_1_1_course}{Course}} happens a few times week. Those times are called slots. Each slots has a fixed group of \mbox{\hyperlink{classtattletale_1_1_actor}{Actors }} that visit the \mbox{\hyperlink{classtattletale_1_1_course}{Course}} during that time. As \mbox{\hyperlink{classtattletale_1_1_actor}{Actors }} can have the same \mbox{\hyperlink{classtattletale_1_1_course}{Course}} a few times each week, some slots feature the same group of \mbox{\hyperlink{classtattletale_1_1_actor}{Actors }} but it is guaranteed that \mbox{\hyperlink{classtattletale_1_1_actor}{Actors }} always visit a single \mbox{\hyperlink{classtattletale_1_1_course}{Course}} in the same group.
