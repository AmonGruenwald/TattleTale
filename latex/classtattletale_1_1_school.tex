\doxysubsection{Detailed Description}
\doxylink{classtattletale_1_1_school}{School} class that is responsible for managing actors and courses during the simulation. 

The \doxylink{classtattletale_1_1_school}{School} class uses a \doxylink{structtattletale_1_1_setting}{Settings } object in its constructor to setup the simulation, instancing the appropriate amount of \doxylink{classtattletale_1_1_actor}{Actors } and \doxylink{classtattletale_1_1_course}{Courses } and then filling said \doxylink{classtattletale_1_1_course}{Courses } with \doxylink{classtattletale_1_1_actor}{Actors }. Afterwards it can be used to run the simulation.
