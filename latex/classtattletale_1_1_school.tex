\doxysubsection{Detailed Description}
\mbox{\hyperlink{classtattletale_1_1_school}{School}} class that is responsible for managing actors and courses during the simulation. 

The \mbox{\hyperlink{classtattletale_1_1_school}{School}} class uses a \mbox{\hyperlink{structtattletale_1_1_setting}{Settings }} object in its constructor to setup the simulation, instancing the appropriate amount of \mbox{\hyperlink{classtattletale_1_1_actor}{Actors }} and \mbox{\hyperlink{classtattletale_1_1_course}{Courses }} and then filling said \mbox{\hyperlink{classtattletale_1_1_course}{Courses }} with \mbox{\hyperlink{classtattletale_1_1_actor}{Actors }}. Afterwards it can be used to run the simulation.
