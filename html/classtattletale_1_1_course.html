Detailed Description

A Course grouping Actors together during different slots of the day.

A Course happens a few times week. Those times are called slots. Each slots has a fixed group of Actors that visit the Course during that time. As Actors can have the same Course a few times each week, some slots feature the same group of Actors but it is guaranteed that Actors always visit a single Course in the same group.
