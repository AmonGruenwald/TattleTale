Detailed Description

Represents an Emotion an Actor has.

Will be instantiated inside a map connecting different EmotionTypes to different Emotions for each Actor.

EmotionType tale::Emotion::StringToEmotionType ( std::string  emotion_string)

Converts a string to an EmotionType.

Uses the same string values EmotionTypeToString returns. So valid strings are: "happy" "calm", "satisfied", "brave" and "extroverted". Everything else just return EmotionType::kNone. (But an assert will crash the Application in Debug mode.)
