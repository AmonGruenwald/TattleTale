Detailed Description

Represents an Emotion an Actor has.

Will be instantiated inside a vector connecting different EmotionTypes (converted to indices) to different Emotions for each Actor.

◆ StringToEmotionType()

EmotionType tattletale::Emotion::StringToEmotionType ( std::string  emotion_string)
static

Converts a string to an EmotionType.

Uses the same string values formater returns. So valid strings are: "happy" "calm", "satisfied", "brave" and "extroverted". Everything else just return EmotionType::kLast. (But an assert will crash the Application in Debug mode.)

Parameters
emotion_stringThe string we want to convert.

The corresponding EmotionType value.
