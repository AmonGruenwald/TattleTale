Detailed Description

Represents a Relationship an Actor has with another Actor.

Will be instantiated inside a two dimensional map connecting different other Actors to RelationshipTypes and corresponding Relationships for each Actor.

Converts a string to an RelationshipType.

Uses the same string values RelationshipTypeToString returns. So valid strings are: love", "attraction", "friendship", "anger" and "protective". Everything else just return RelationshipType::kNone. (But an assert will crash the Application in Debug mode.)
