Detailed Description

Represents a Relationship an Actor has with another Actor.

Will be instantiated inside a map connecting different other Actors to a Vector containing Relationships for each Actor mapped to indices converted from their RelationshipType.

RelationshipType tattletale::Relationship::StringToRelationshipType ( std::string  relationship_string)

Converts a string to an RelationshipType.

Uses the same string values formater returns. So valid strings are: love", "attraction", "friendship", "anger" and "protective". Everything else just return RelationshipType::kLast. (But an assert will crash the Application in Debug mode.)
