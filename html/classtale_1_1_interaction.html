Detailed Description

Represents an Interaction an Actor had with other Actors .

Extends the Kernel class, so it has to hold every information that is necessary to recreate the causal chain leading to this Event.

◆ relationship_effects_

std::vector<std::map<size_t, std::map<RelationshipType, float> > > tale::Interaction::relationship_effects_

Stores the effect this Interaction has on each participating Actor's Relationships .

For each Actor a map is created that maps to participants index in the participants_ vector another map of RelationshipType float pairs. Those pairs signal how much each Relationship axis would be changed.
