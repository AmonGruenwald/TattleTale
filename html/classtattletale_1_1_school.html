Detailed Description

School class that is responsible for managing actors and courses during the simulation.

The School class uses a Settings object in its constructor to setup the simulation, instancing the appropriate amount of Actors and Courses and then filling said Courses with Actors . Afterwards it can be used to run the simulation.

Finds a random group of Actors that could take the Course during those slots.

Picks random Actors and checks if they do not already have the Course corresponding to the course_id. It furhter makes sure that the Actors also are free during the passed slots. Because of those conditions it is not guaranteed that a course_group will have the full amount of Actors specified in setting_.

Parameters
course_idThe id of the course we ant to find a a random group for. Corresponds to the courses_ vector.
slotsThe slots we want to find free Actors for.
Returns
The found group of Actors.

Transforms a Weekday and a daily tick to a slot index.

Slots increase each day. So if we assume there are 6 slots per day, the slot indices for monday are 0, 1, 2, 3, 4, 5 and the slots for tuesday 6, 7, 8, 9, 10, 11. This transforms a Weekday and daily tick index (which in the example above would go from 0-5) to corresponding slot index.

Parameters
weekdayThe Weekday from which we want to transform the daily tick to a slot index.
daily_tickThe daily tick we want to transform to a slot index.
Returns
The slot index.
