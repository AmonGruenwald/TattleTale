◆ CalculateInteractionChance()

float tattletale::Actor::CalculateInteractionChance ( const InteractionTendency &  tendency,
const ContextType &  context,
Kernel *&  out_reason 
)

Calculates the chance of an Interaction based on its InteractionTendency and the Actor's current state.

Every value used is based between -1.0 and 1.0 except the return value which is between 0.0 and 1.0. Because the Actor uses this value for a distribution to pick the Interaction it wants to do, 1.0 does not automatically mean an Interaction has a 100% chance of being picked. If we have for example three Interactions , which prodcue the retun values 1.0, 1.0 and 0.0. Then the first two Interactions have a chance of 50% each and the last one has a 0% chance.

Parameters
[in]tendencyThe tendency of the Interaction we want the chance for.
[in]contextIn which context the Interaction would happen.
[in]group_size_ratioHow big the group is in which this Interaction is happening (between -1.0 and 1.0).
[out]out_reasonWhich part of the state of the Actor increased the chance the most.
Returns
A chance value between 0.0 and 1.0.

◆ ChooseInteraction()

int tattletale::Actor::ChooseInteraction ( const std::list< Actor * > &  actor_group,
ContextType  context,
std::vector< Kernel * > &  out_reasons,
std::vector< Actor * > &  out_participants,
float &  out_chance 
)

Let's the actor decide on an Interaction for the currrent situation depending on their internal state.

This first weeds out Interaction that are not possible due to their InteractionRequirements. Then calculates chances for each possible one based on their respective InteractionTendency and the Actor's internal state and then chooses one at semi randomly based on the chances. Afterwards a similar process is employed to pick necessary participants for the Interaction.

Parameters
[in]actor_groupThe course group the Actor is currently interacting in.
[in]contextThe ContextType this Interaction will be in.
[out]out_reasonsVector the Actor will write it's reason for the decision to.
[out]out_participantsVector the Actor will to with which other Actors he wants to do the Interaction.
[out]out_chanceHow likely it was that this interaction was chosen.
Returns
The index of the InteractionPrototype the Actor chose.

◆ GetFreetimeActorGroup()

std::list< Actor * > tattletale::Actor::GetFreetimeActorGroup ( ) const

Returns a list of Actors this Actor has the strongest Relationship with. The size of this list is determined by the Setting.

This group of Actors will be used for Interaction that happen during the freetime of the Actor. This happens because it makes sense for the Actor to interact with those Actors he has the strongest feelings for (be they negative or positive).

Returns
The list of Actors .
